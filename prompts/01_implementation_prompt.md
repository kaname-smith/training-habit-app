# Claude Code 実装開始プロンプト

実装開始してください。

範囲は `docs/10_development_roadmap.md` の Phase 1 とします。

必須要件：

- React + TypeScript + Vite を基本構成にする。既存構成がある場合は既存構成を尊重する。
- スマホ第一のレスポンシブUIにする。
- 初期設定、今日のダッシュボード、トレーニング実行、記録カレンダー、プロテイン/栄養、設定画面を実装する。
- 7/7〜7/31の試験月用プログラムを初期データとして入れる。
- localStorageで記録が残るようにする。
- 10分版メニュー、Aメニュー、Bメニュー、休息を切り替えられるようにする。
- 痛み・体調不良時の中止注意をUIに入れる。
- 型チェック、リント、テストを整える。

実装後に必ず次を報告してください。

1. 追加・変更したファイル
2. 起動方法
3. 確認方法
4. 実行したチェック
5. 未完了事項
