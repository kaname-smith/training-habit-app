# ui-review

Review the current UI against `docs/03_ui_ux_spec.md`.

Focus on:

- smartphone-first use
- clear first-view recommendation
- low cognitive load during exam season
- positive handling of 10-minute workouts and rest
- accessibility
- clean visual hierarchy

Fix high-impact UI issues and then report what changed and how to verify it.
