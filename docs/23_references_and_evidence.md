# 23 References and Evidence

## 1. 身体活動・筋力トレーニング

### World Health Organization

- Physical activity
  - https://www.who.int/initiatives/behealthy/physical-activity
- Physical activity fact sheet
  - https://www.who.int/news-room/fact-sheets/detail/physical-activity
- WHO guidelines on physical activity and sedentary behaviour
  - https://www.who.int/europe/publications/i/item/9789240014886

アプリでの利用：

- 少しでも身体活動を行う考え方
- 成人の中強度活動量の一般目安
- 主要筋群の筋力強化を週2日以上行うという一般目安

### 厚生労働省

- 健康づくりのための身体活動・運動ガイド2023
  - https://www.mhlw.go.jp/content/001194020.pdf
- 身体活動・運動の推進
  - https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/undou/index.html
- 筋力トレーニングについて
  - https://www.mhlw.go.jp/content/001195869.pdf

アプリでの利用：

- 成人の筋力トレーニング週2〜3日
- 自重トレーニングも筋力トレーニングに含まれる
- 個人差を踏まえ、可能なものから取り組む
- 座位時間を長くしすぎない

## 2. 栄養・タンパク質

### 厚生労働省

- 日本人の食事摂取基準（2025年版）
  - https://www.mhlw.go.jp/stf/newpage_44138.html
- 日本人の食事摂取基準の一覧
  - https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/kenkou/eiyou/syokuji_kijyun.html

### International Society of Sports Nutrition

- Position Stand: protein and exercise
  - https://pubmed.ncbi.nlm.nih.gov/28642676/

アプリでの注意：

- 政府の食事摂取基準と、運動者向けスポーツ栄養の提案を同じものとして扱わない
- 1.4〜1.6g/kg/dayは全成人への必須推奨量ではなく、トレーニング利用者向けの実用設定として説明する
- 腎疾患等の個別条件についてアプリが判断しない

## 3. Google Calendar

### Google公式

- Google Calendar API overview
  - https://developers.google.com/workspace/calendar/api/guides/overview
- Choose Google Calendar API scopes
  - https://developers.google.com/workspace/calendar/api/auth
- Using OAuth 2.0 to Access Google APIs
  - https://developers.google.com/identity/protocols/oauth2

アプリでの利用：

- Calendar APIはREST API
- ユーザーカレンダーへのアクセスにはOAuth 2.0が必要
- 必要最小限のscopeを選ぶ
- 初期版では手入力またはICS importを優先する
