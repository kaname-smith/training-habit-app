## 3. このアプリ内で使う考え方

- 成人の運動習慣では、週150分程度の中強度運動と、週2回以上の主要筋群の筋力トレーニングが一般的な健康目安になる。
- 運動者のタンパク質摂取は、1.4〜2.0g/kg/day が一般的な参考範囲になる。
- 7月は試験月であるため、このアプリでは 1.4〜1.6g/kg/day を現実的な初期目標として扱う。
- アプリは医療判断をしない。痛み・体調不良時は運動中止を促す。

## 4. 実装上の注意

- 参照資料はアプリ内に長文転載しない。
- アプリ内では簡潔な説明に留める。
- 健康情報は断定しすぎない。
- 個人差を前提にする。
