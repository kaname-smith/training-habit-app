# 02 Training Recipe July — 7/7夜〜7/31のトレーニング内容

## 1. 今月の原則

7月は期末試験月である。目的は、筋肉を限界まで追い込むことではなく、8〜9月の本格トレーニングに入るための土台を作ることである。

基本原則：

- 疲労を残さない。
- 毎回「あと2〜3回できる」余裕を残す。
- マンション自室でできる静かな種目にする。
- ジャンプ系、足音が出るHIITは避ける。
- 試験前は10分版または休息でよい。
- 睡眠不足の日は軽くする。

## 2. 7/7夜の初期メニュー

初日は開始の儀式とする。

| 種目 | 回数 |
|---|---:|
| スクワット | 10回 |
| 膝つき腕立て伏せ | 8回 |
| グルートブリッジ | 12回 |
| プランク | 20秒 |
| 肩回し・股関節回し | 2分 |

アプリでは「初期設定後に10分だけ動く」という導線にする。

## 3. 週間スケジュール

基本は週3回。

| 曜日 | 内容 | 時間 |
|---|---|---:|
| 月 | 筋トレA | 25分 |
| 火 | 歩行・ストレッチ | 10〜20分 |
| 水 | 筋トレB | 25分 |
| 木 | 休み or 散歩 | 10分 |
| 金 | 筋トレA | 25分 |
| 土 | 軽い有酸素・散歩 | 20〜40分 |
| 日 | 休み・睡眠優先 | - |

7/7が火曜夜であるため、初期スケジュールは以下にする。

| 日付 | 内容 |
|---|---|
| 7/7 火 夜 | 初期設定＋10分導入 |
| 7/8 水 | 筋トレA |
| 7/10 金 | 筋トレB |
| 7/12 日 | 筋トレA or 軽め |
| 7/13〜7/19 | A/B/Aを継続 |
| 7/20〜7/26 | 試験が重い日は10分版 |
| 7/27〜7/31 | 疲労を残さない維持モード |

アプリでは日付固定ではなく、開始日から自動生成できる構造にする。ただし、初期データとして7/7〜7/31を持たせる。

## 4. 筋トレA

目的：下半身、胸、背中、体幹。

| 種目 | 回数・時間 | セット | フォーム注意 |
|---|---:|---:|---|
| スクワット | 12〜15回 | 2〜3 | 膝とつま先を同じ方向へ。反動を使わない。 |
| 腕立て伏せ | 6〜12回 | 2〜3 | きつければ膝つき。体幹を一直線に保つ。 |
| グルートブリッジ | 15回 | 2 | 腰を反りすぎず、お尻を締める。 |
| バックパックロー | 10〜15回 | 2 | 背中を丸めず、肩甲骨を寄せる。 |
| プランク | 20〜40秒 | 2 | 腰を落とさない。息を止めない。 |
| デッドバグ | 左右10回 | 2 | 腰が浮かない範囲で動かす。 |

## 5. 筋トレB

目的：脚、肩、背中、腹筋。

| 種目 | 回数・時間 | セット | フォーム注意 |
|---|---:|---:|---|
| スプリットスクワット | 左右8〜10回 | 2 | 前足で床を押す。膝が内側に入らない。 |
| パイクプッシュアップ | 6〜10回 | 2 | 肩に効かせる。きつければ通常腕立てに変更。 |
| ヒップヒンジ | 12〜15回 | 2 | 背中を丸めず、お尻を後ろに引く。 |
| バックパックロー | 10〜15回 | 2〜3 | リュックを体に近づけて引く。 |
| サイドプランク | 左右20〜30秒 | 2 | 体を一直線に保つ。 |
| カーフレイズ | 15〜20回 | 2 | ゆっくり上下する。 |

## 6. 10分版

疲れている日、試験直前、時間がない日のメニュー。

| 種目 | 回数 |
|---|---:|
| スクワット | 15回 |
| 腕立て伏せ | 8回 |
| バックパックロー | 12回 |
| グルートブリッジ | 15回 |
| プランク | 30秒 |

1〜2周で終了。

アプリでは「10分だけやる」をポジティブな選択肢として扱う。代替メニューではなく、試験月の正式メニューとする。

## 7. 進歩ルール

毎回、次のどれか1つだけ進める。

| 状態 | 次にやること |
|---|---|
| 余裕がある | 回数を2回増やす |
| 回数上限までできた | セットを1つ増やす |
| セットも余裕 | リュックに本を1〜2冊追加 |
| 疲れている | セットを減らす |
| 試験前 | 10分版にする |

## 8. タンパク質・プロテイン

7月の実用目標：

```text
1日のタンパク質目標 = 体重 × 1.4〜1.6g
```

表示例：

| 体重 | 1日タンパク質目標 |
|---:|---:|
| 55kg | 77〜88g |
| 60kg | 84〜96g |
| 65kg | 91〜104g |
| 70kg | 98〜112g |

プロテイン摂取の基本：

- トレーニング後：20〜30g
- 朝食が弱い日：20g程度
- 食事で不足した日：夜に20g程度

プロテインは食事の代替ではなく、不足分を埋める道具として扱う。
